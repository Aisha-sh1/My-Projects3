# Responsive Animal Slider

A lightweight, fully responsive image slider built from scratch using pure frontend web technologies. This project demonstrates smooth UI transitions and dynamic layout adjustments without relying on external libraries or No-Code platforms.

## Features
* **Pure Web Stack:** Built exclusively with vanilla HTML, CSS, and JavaScript.
* **Fully Responsive:** Adapts seamlessly to various screen sizes and device aspect ratios.
* **Dynamic Navigation:** Smooth slide transitions driven by JavaScript event listeners.
* **Clean UI Architecture:** Structured layouts utilizing CSS Flexbox/Grid and precise positioning.

## Technologies Used
* **HTML5:** Semantic structure of the slider container and components.
* **CSS3:** Custom styling, layout design, and smooth transitional animations.
* **JavaScript (ES6):** DOM manipulation to handle next/previous navigation logic.
